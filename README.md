# 🛒 Análise de Dados de Clientes — Store 1

Projeto desenvolvido durante o Sprint 01 do bootcamp **TripleTen** de Análise de Dados.

## 📋 Objetivo

Avaliar a qualidade de uma amostra de dados de clientes de um e-commerce e prepará-los para análises futuras, aplicando técnicas de limpeza e manipulação de dados com Python.

## 🔍 O que foi feito

- Identificação e correção de tipos de dados incorretos (`float` → `int`, `string` → `int`)
- Limpeza de strings: remoção de espaços e substituição de caracteres
- Manipulação de listas e categorias de produtos
- Cálculo de métricas de gastos (total, mínimo e máximo)
- Uso de estruturas de controle (`while`, `for`) para simular compras
- Criação de strings formatadas com `f-string`
- Cálculo de receita total a partir de listas aninhadas

## 🛠️ Ferramentas utilizadas

- Python 3
- Jupyter Notebook
